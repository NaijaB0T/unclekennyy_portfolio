Favicon Generator Instructions

To create a proper favicon.ico file:

  1. Use a favicon generator service like favicon.io or realfavicongenerator.net
  2. Upload the SVG file we've created
  3. Download the generated favicon package
  4. Place the favicon.ico file in the public folder of your Next.js project
